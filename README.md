# 笑话大全网站

一个有趣的笑话网站，每天为您带来欢乐！

## 功能特点

- 🎭 随机笑话展示
- 📱 响应式设计
- 🎨 美观的界面
- ⚡ 快速加载

## 技术栈

- HTML5
- CSS3
- JavaScript
- Vercel部署

## 本地运行

直接在浏览器中打开 `index.html` 文件即可运行。

## 部署

本项目已配置为在Vercel上自动部署。

### Vercel部署步骤：

1. 确保代码已推送到GitHub仓库
2. Vercel会自动检测到仓库变化
3. 自动构建和部署
4. 访问生成的域名即可查看网站

## 项目结构

```
joke-website-1/
├── index.html      # 主页面
├── package.json    # 项目配置
├── vercel.json     # Vercel部署配置
└── README.md       # 项目说明
```
